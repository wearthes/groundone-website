# Ground One rebuilt website package

## Structure
- `index.html` = Home
- `coffee.html` = Coffee
- `business.html` = Business
- `safari.html` = Safari
- `styles.css` = shared styles
- `script.js` = mobile menu
- `assets/` = logo and imagery

## What changed
- Home now carries the brand "why"
- Coffee now has its own dedicated page
- About removed from the navigation
- One clear Shop button in the header on every page
- Shopify is treated as the store path, not duplicated inside the site structure

## Publish steps
1. Upload all files and the `assets` folder to your site root.
2. Replace any older files with these versions.
3. Clear cache / hard refresh after publish.
4. Check that all Shopify buttons open the correct store.
